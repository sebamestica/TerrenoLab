# Data Samples - TerrenoLab

This directory contains topographic datasets for testing the TerrenoLab import and processing features.

## Files

- `terreno_prueba.csv`: A synthetic survey containing 50 points in UTM coordinates.
  - Coordinate system: Projected Grid (UTM)
  - Columns: `Este` (Easting/X), `Norte` (Northing/Y), `Cota` (Elevation/Z)
  - Range: ~500m x 500m bounding area, sloped towards the northeast (124m to 152m).
